This app is a simple blog built using Python's Flask framework. The app allows for the user to "post" new blogs to the page through the command line, and keeps a list of all blogs written, with a clickable Title link that redirects to the blog page and a subtitle. Please excuse the mediocre writing in the blogs; the posts are there merely for demonstration purposes. Most are simply ramblings about my favorite animated shorts which I am most passionate about.

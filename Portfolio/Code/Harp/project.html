Harp : Java

This program transforms the user's keyboard into a harp. As notes are played, circles are drawn on the screen to provide a visual effect. The harp sound is simulated using a ring buffer data structure and the Karplus-Strong algorithm. The visuals are provided using the PennDraw library, which can be found here. The notes range from Concert A (440.0 Hz), which corresponds to 'q', to the A three octaves above it, which corresponds to ' ' (space bar).

The first video in the playlist shows all the notes typed once in ascending order. In the Interactions panel, there is a live feed of which note is being pressed to go along with the sound and visual.

The second video shows a sample of a song being played on the digital harp.

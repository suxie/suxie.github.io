It's Raining Cats + Dogs : Java

This is a game I came up with myself, entitled "It's Raining Cats + Dogs," written using Java AWT and Java Swing. The premise of the game is as follows: before each game, the user is first prompted to choose "cats or dogs?". Based on their response, their goal in the game is to catch either the falling cats or the falling dogs. There are "normal" animals, but also special animals with names like "Santa Paws" and "Jurassic Bark." To gain points, the user must catch their animal of choice. Normal animals are worth +10 points each, and special animals are worth more or less depending on their speed of falling. The user moves around a basket at the bottom of the screen using the left and right arrow keys. If a normal animal is dropped or the user catches the wrong animal, the user loses a life (everyone starts out with 3). Special animals are considered "bonuses," and get no penalty when dropped.

At the end of each game, a screen showing all the special animals encountered appears, and the user is redirecteds to a high score table. To actually implement all these features, high scores were treated as comparables, and I had an interface "Animal" as well as two abstract classes for "Cat"s and "Dog"s. To allow the user to interact with the game, I used Java AWT's event listeners, and to display all the graphics and text, I used Java Swing. All screens and assets were drawn in Photoshop and displayed as images.

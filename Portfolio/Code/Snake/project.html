Snake : Java

This program is a version of the classic game, Snake, in which a snake controlled by the user's keyboard is navigated around the screen ('A' moves left, 'D' moves right, 'W' moves up, and 'S' moves down) to eat a series of bits. The user loses if the snake crashes into itself or the edge of the screen. The graphical interface is created with the PennDraw library, available here. The snake always starts out moving towards the right from a random position on the left side of the screen, and the food is always randomly generated. A high score table is shown at the beginning of the game with dummy scores, and as the user plays, new high scores are added so that the top high scores are always displayed in descending order.

Each point is drawn as a "bit" object, and these bits are combined in an array list to form a "snake" object. The "gameboard" is treated as a grid, where the snake's body and the food being consumed is treated as points on the grid. High scores are sorted using comparators and the Java Collections library.

The video shows the game being played with all of the audio-visual components that were added for greater game enjoyment. A live feed of the keys being typed can be seen in the Interactions panel in the Java compiler.
